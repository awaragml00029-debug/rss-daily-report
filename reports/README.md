# 示例报告

这个目录用于存放生成的报告。

## 目录结构

```
reports/
├── 2025/
│   ├── 10/
│   │   ├── daily-2025-10-29.md
│   │   ├── daily-2025-10-30.md
│   │   └── daily-2025-10-31.md
│   ├── 11/
│   │   └── ...
│   ├── monthly-2025-10.md
│   └── monthly-2025-11.md
└── 2026/
    └── ...
```

## 命名规则

- 每日报告：`daily-YYYY-MM-DD.md`
- 月度报告：`monthly-YYYY-MM.md`

## 注意事项

- 报告由 GitHub Actions 自动生成
- 不要手动编辑报告文件
- 所有报告都有 Git 版本控制
